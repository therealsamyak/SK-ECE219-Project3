## Project Overview

This project is organized into three complementary parts that together illustrate how small language models can be adapted, integrated, and deployed for real-world data mining tasks.

- **Part A:** Improve reasoning performance of language models through parameter-efficient fine-tuning and prompting strategies.
- **Part B:** Build a structured ReAct-style agent that performs data analysis over real CSV tables with bounded context and reliable outputs.
- **Part C:** Perform regression analysis and feature engineering, integrating classical statistical modeling with agent-assisted automation.
